# プライバシーポリシー (Privacy Policy)

ExcelBench（以下「当プロジェクト」といいます）は、本ソフトウェアの提供にあたり、ユーザーの情報の取り扱いについて以下の通り定めます。

## 1. 収集する情報
本ソフトウェアは、以下の情報を収集・送信する場合があります。
- **システム構成情報**: CPU名、メモリ容量、ストレージの種類、Excelのバージョン、OSの種類。
- **ベンチマーク結果**: 計測にかかった時間、スコア。
- **IPアドレス**: 居住国判別（ipapi.co）およびGitHubからの設定取得のために使用されます。

## 2. 利用目的
収集したデータは以下の目的で使用されます。
- アプリケーション内の統計表示（世界平均との比較）。
- 居住国に応じた最適な推奨PC（アフィリエイトリンク）の提示。
- ソフトウェアの改善および不具合の修正。

## 3. データの匿名性
Google Forms等へ送信されるデータはすべて匿名化されており、特定の個人を直接識別できる情報は含まれません。

## 4. 第三者への提供
法令に基づく場合を除き、取得した個人情報をユーザーの同意なく第三者に提供することはありません。居住国の判別、外部サービスのAPI（ipapi.co等）を使用する場合があります。

## 5. 免責事項
外部サービス（Google, ipapi等）の利用規約およびプライバシーポリシーについては、それぞれのサービスの定めるところによります。

## 6. お問い合わせ
本ポリシーに関するお問い合わせは、GitHubのリポジトリのIssue等を通じて受け付けております。
